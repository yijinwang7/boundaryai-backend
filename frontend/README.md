# Survey Frontend

A minimalist React frontend for creating and managing surveys with dynamic questions and options.

## Prerequisites

Built on :
* Node.js v24.3.0
* npm 11.4.2

## Quick Start

1. Install dependencies:

   ```bash
   npm install
   ```
2. Run in development:

   ```bash
   npm start
   ```
3. Build for production:

   ```bash
   npm run build
   ```

## Features

* Add, edit, and delete questions
